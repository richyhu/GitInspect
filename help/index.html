如何克隆仓库

你可以通过 HTTPS 或 SSH 两种方式克隆仓库。填写仓库信息后，可一键复制对应命令。

HTTPS 方式
git clone https://github.com/<owner>/<repo>.git
SSH 方式
git clone git@github.com:<owner>/<repo>.git
提示：在主页面中搜索 owner/repo 后，顶部“克隆信息”区域也提供复制按钮与 HTTPS/SSH 切换。

如何添加 GitHub API 令牌

  1. 打开 GitHub → 右上角头像 → Settings（设置）。
  2. 左侧导航最下方进入 Developer settings → Personal access tokens。
  3. 选择 Fine-grained tokens 或 Tokens (classic) 并创建令牌。
  4. 若需要访问私有仓库，请在 Classic 模式勾选 repo；仅公共仓库可勾选 public_repo。
  5. 生成令牌后，复制保存（GitHub 只显示一次）。
  6. 返回主页面，找到 “GitHub API 令牌” 输入框，粘贴令牌并点击 保存 或 测试令牌。
令牌保存在浏览器的本地存储中，只有你自己可见。若遇到访问限流（如 403），建议添加令牌后重试。
